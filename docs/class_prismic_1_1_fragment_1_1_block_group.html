Detailed Description

This class embodies a group of StructuredText blocks. This is only for internal use, when a StructuredText fragment gets rendered as HTML: the first thing that happens is that StructuredText blocks get grouped together, so that list-items can later be serialized with a

and a after, for instance. This is the data structure that allows that temporary storage.
