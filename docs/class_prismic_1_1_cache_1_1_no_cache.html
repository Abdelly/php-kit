Detailed Description

A cache implementation that doesn't cache anything; to be passed as the $cache parameter of Prismicget when you don't want any caching. This documentation right here introduces what the functions are supposed to do if there is a cache involved, even though in this class in particular, they all simply do nothing and return false.
