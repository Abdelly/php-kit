Detailed Description

This class embodies an emphasize span (typically rendered as italic). A span comes in a array of spans, which is served with a raw text. If the raw text is "Hello world!", and the EmSpan's start is 6 and its end is 11, then the piece that is meant to be emphasized is "world".
