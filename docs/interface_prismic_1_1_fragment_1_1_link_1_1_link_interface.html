Detailed Description

This interface embodies any link. LinkInterface objects can be found in two occasions: as the "$link" variable of a HyperlinkSpan object (which happens when the link is a hyperlink in a StructuredText fragment), or the LinkInterface can also be its own fragment (e.g. for a "related" fragment, that links to a related document).

Its known implementations are DocumentLink, WebLink, and also FileLink and ImageLink (united as "MediaLink" in former versions of the kit).
