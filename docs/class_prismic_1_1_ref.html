Detailed Description

Embodies a ref to be called on the prismic.io repository. The ref is a prismic.io concept that represents a time on which you wish to query the repository, in the present (the content that is live now, we call this ref the master ref) or in the future (the content that is planned for a future content release).

This is meant to be built during the unmarshaling of the /api document, and is not meant to be used externally except for testing.
